Open Source NFT Operating System get started


Nuron is an open source and self-hosted server that takes HTTP requests and turns them into NFTs.

Self-hosted

No need to rely on someone else's service or contract. Directly interact with the blockchain and IPFS to build NFTs.

Easy

No need to learn Solidity, IPFS, and all the complicated tech. Nuron takes care of everything behind the scenes. All you need to know is JavaScript.

Free

Create as many NFTs as you want for FREE. You don't need to spend any transaction fee. It's the minters who pay when minting.
